## Install

```bash
npm install @filipemeneses/unity-to-json
```

```bash
yarn add @filipemeneses/unity-to-json
```

## Usage

```ts
import { myPackage } from '@filipemeneses/unity-to-json';

myPackage('hello');
//=> 'hello from my package'
```
